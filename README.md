## Build 

Build latex-document:

```
git clone https://github.com/tobiasanker/Sakura-Project-Documentation.git
cd Sakura-Project-Documentation
make latexpdf 

# pdf-document is generated under ./build/latex/Sakura-Project.pdf
```
